# 🤖 AI Resume Screening Agent

An AI-powered Resume Screening Agent built with **n8n**, **Groq LLM**, **Google Sheets**, and **Gmail API**.

This workflow automatically analyzes candidate resumes, compares them with a job description, scores candidates, stores results in Google Sheets, and sends automated email notifications.

## 🚀 Features

- 📄 Resume PDF Upload
- 🤖 AI Resume Analysis using Groq LLM
- 🎯 Job Description Matching
- 📊 Candidate Scoring (0–100)
- ✅ Automatic Shortlist / Reject Decision
- 📋 Save Candidate Data to Google Sheets
- 📧 Automated Email Notifications

## 🛠️ Tech Stack

- n8n
- Groq LLM
- Google Sheets API
- Gmail API
- PDF Parser
- AI Agent

## 📂 Workflow

Resume Upload Form
→ PDF Extraction
→ AI Resume Analysis
→ Candidate Scoring
→ Conditional Decision
→ Google Sheets
→ Email Notification
